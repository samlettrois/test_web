Q? io 도메인은 무엇인가

본래 io는 영국령 인도양 지역(British Indian Ocean)의 약자다.
즉, 최상위 도메인에 해당하는 국가코드인 것인데 영국령 인도양 지역과 상관없는 사이트들도 io 도메인을 사용하게 된 것일까?
A! IT 기업이 하는 일과 일맥상통하는 생김새와 넉넉한 도메인 자리 덕에 IT 스타트업들이 io 도메인을 사용하기 시작했고, 결국 유행처럼 번져 대다수 IT 관련 스타트업들이 사용하게 되었다는 것이다.

Q? localStorage와 JSON의 직렬화

A!
- JSON(JavaScript Object Notation)
 데이터를 전달할 때 JS가 그걸 다룰 수 있도록 Object로 바꾸어줌.
 즉, Object->String or String ->Object

- localStorage에 저장되는 값들(key:value)은 다 String으로 저장된다.
 true라고 value값을 줬어도 "true"라고 스트링으로 저장되는 것이다.

- localStorage는 객체 또는 배열을 허용하지 않는다.
 하지만 JSON은 API를 사용해 직렬화해서 저장할 수 있다.

ex) localStorage.setItem(TODOS_LS, JSON.stringify(toDos));
  //JSON.stringify(): JS Object를 String으로 바꿔주는 함수.
  //toDOs는 배열값이지만, JSON으로 직렬화되어 TODOS_LS의 value에 저장되는 것

ex) const loadedToDos = localStorage.getItem(TODOS_LS);
  const parsedToDos = JSON.parse(loadedToDos);
  //Sting으로 저장되어 있는 loadedToDos값을 Object값으로 변환하여 준다.
